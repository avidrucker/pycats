## Notes

- Some kaomoji rely on Unicode combining characters and may render differently across terminals.
- Cat faces using `･`, `ω`, `◕`, `ᴥ`, `ↀ` need UTF-8 + font support for the best experience.
- The table-flip `(╯°□°)╯︵ ┻━┻` is included for completeness but is not cat-themed.
